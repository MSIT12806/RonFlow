# Notes

這個目錄用來放 RonFlow 的個人思考、方法論偏好、設計價值觀與開發規範。

它與 `docs/tech-base` 的差別是：

- `docs/tech-base`：偏向名詞解釋、概念說明、通用知識索引。
- `docs/notes`：偏向作者立場、判斷原則、設計取向、命名慣例與會後反思。
