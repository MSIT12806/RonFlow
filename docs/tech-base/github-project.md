# GitHub Project

GitHub Project 是 GitHub 提供的專案管理功能，可以用表格、看板或時間軸等形式，整理 repository 裡的工作項目。它常被拿來管理功能需求、技術工作、缺陷、文件更新與里程碑規劃。

它的價值不只是把項目排成一列，而是能把 [Backlog](./backlog.md)、工作狀態與追蹤欄位集中在同一個地方，讓專案管理不必完全散落在筆記、Issue 與聊天訊息中。

如果團隊採用 [Kanban](./kanban.md) 或自訂 [Workflow](./workflow.md)，GitHub Project 也能拿來呈現欄位狀態、追蹤 WIP，甚至依欄位或標籤做出不同視圖。對個人專案來說，它是一個相對低成本但很實用的管理入口。
