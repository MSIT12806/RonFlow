# Project Member

Project Member 指的是被視為某個專案工作流一部分、可以參與任務協作的人。它是一種相對泛化的業務角色，用來表示「這個人屬於專案成員」，但不預設他一定是哪一種專職身分。

在專案管理工具裡，Project Member 可能有建立任務、移動狀態、調整優先順序或補充資訊等能力；至於具體可以做到哪一步，通常還要再看權限規則或團隊約定。

因此，Project Member 比較像是參與專案的通用角色，而不是某個特定職能，也不一定等於 [Assignee](./assignee.md) 或 [Current Action Owner](./current-action-owner.md)。
