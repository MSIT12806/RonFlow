# Domain Expert

Domain Expert 指的是對某個業務領域具有真實經驗與判斷能力的人。他不一定寫程式，但知道流程如何運作、哪些規則是現場真正在意的，以及哪些例外情境會影響決策。

在 [Event Storming](./event-storming.md)、需求分析或 [Domain Modeling](./domain-modeling.md) 過程中，Domain Expert 的價值在於校正語意。當團隊討論某個事件、規則或流程時，真正能確認「這在業務上是不是這樣」的人，往往不是工程師，而是 Domain Expert。

因此，Domain Expert 不只是提供背景資料的人，也是在建模時用來避免語意失真的重要參與者。
