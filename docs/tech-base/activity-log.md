# Activity Log

Activity Log 指的是系統對操作歷程的活動記錄，用來追蹤某個人或某個系統在什麼時間做了什麼事。它常見於任務系統、後台管理或協作工具中，幫助使用者回看一筆資料曾經歷過哪些互動與變動。

這類記錄可能包含欄位更新、指派變更、排序調整、留言、狀態切換或其他操作痕跡。它的重點通常是可追溯性與可閱讀性，而不是要表達每一筆紀錄都具有完整的業務語意。

因此，Activity Log 不一定等於 [Domain Event](./domain-event.md)。有些活動記錄只是操作軌跡，對業務規則本身沒有那麼強的語意重量；但它仍然可能對協作、除錯與溝通非常有價值。
