# Priority

Priority 指的是一項工作和其他工作相比的相對重要性與排序位置。它通常反映的是產品價值、風險、策略目標或團隊目前認為最值得先投入的方向，而不一定代表這件事必須立刻處理。

在專案管理裡，Priority 常被拿來決定 [Backlog](./backlog.md) 的先後順序，幫助團隊知道哪些工作應該先做、哪些工作可以延後。高 Priority 不等於一定緊急，它只是表示這件事在目前整體排序中比較重要。

因此，Priority 很適合和 [Urgency](./urgency.md) 分開看待。某些工作可能很重要，但沒有立即時間壓力；也有些工作雖然臨時而急迫，長期價值卻不一定最高。
