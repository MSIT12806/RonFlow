# .NET

.NET 是微軟維護的開發平台，提供執行環境、基礎類別庫、語言工具鏈與多種應用程式框架。開發者可以用它來建立 Web 應用、桌面程式、背景服務、雲端系統，或其他類型的軟體。

在現代 .NET 生態中，常見語言包括 C#、F# 與 Visual Basic，而很多專案會搭配 ASP.NET Core、Entity Framework Core、測試框架與命令列工具一起使用。對團隊來說，.NET 不只是語言選擇，也會影響專案結構、部署方式與開發習慣。

當文章提到 .NET 架構或 .NET solution 時，通常是在談這個平台下如何組織程式碼、分層、測試與專案邊界，而不只是單指某個執行檔或套件。
